About Pairing

A pairing code allows you to securely link your WhatsApp accounti without scanning a QR code. This is especially useful for headless servers and remote deployments.

How it works:

  • 1 Enter your WhatsApp phone number with the correct country code
  • 2 The system generates a unique 8-digit pairing code for your number
  • 3 Copy the code or open it directly in WhatsApp
  • 4 In WhatsApp, go to Settings → Linked Devices → Link with Phone Number
  • 5 Enter the pairing code to complete the secure connection
