About Pairing

A pairing code allows you to securely link your WhatsApp account without scanning a QR code. This is especially useful for headless servers and remote deployments.

How it works:

  • 1Enter your WhatsApp phone number with the correct country code
  • 2The system generates a unique 8-digit pairing code for your number
  • 3Copy the code or open it directly in WhatsApp
  • 4In WhatsApp, go to Settings → Linked Devices → Link with Phone Number
  • 5Enter the pairing code to complete the secure connection
